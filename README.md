# twitter-pds2

## Dependencias necessarias:

Cmake: https://cmake.org/download/

Sqlite3: https://www.sqlite.org/download.html ou

```sudo apt-get install cmake sqlite3 libsqlite3-dev```

## Instalando:
``` cmake .```\
``` make```\
``` ./TwitterApp migrate``` # Para criar o banco de dados\

## Executando:
``` ./TwitterApp```


## Testando:
``` ./test```
