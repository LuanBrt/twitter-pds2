Twitter-pds2

Esse é um projeto desenvolvido em C++ com o intuito de ser um clone da rede social "X", antigo "Twitter".\ Essa é uma aplicação de linha de comando que permite aos usuários interagirem com uma versão simplificada de uma rede social, similar ao Twitter. Com esta ferramenta, os usuários podem postar tweets, visualizar seu feed e seguir outros usuários.
